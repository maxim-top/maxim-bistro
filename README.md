# Maxim's Bistro
欢迎来到美信拓扑的小酒馆，我们终于有机会一起坐坐，聊聊关于美信拓扑和蓝莺IM的事儿了 :wink:

蓝莺IM，是由[美信拓扑](https://www.maximtop.com/)团队研发的新一代即时通讯云服务，SDK设计简单集成方便，服务采用云原生技术和多云架构，私有云也可按月付费。
